# Fasting Companion

Fasting Companion is a mini project built to support my friend Kike during fasting periods.

The goal is to make fasting guidance simple, visual, and practical: choose a fasting duration, follow hydration recommendations, review safety reminders, and track the fast with a live countdown.

## Project Purpose

This project focuses on solving a real user need:
- safer fasting planning,
- better hydration awareness,
- clear step-by-step flow,
- simple mobile-friendly experience.

## Features

- Multi-step fasting setup flow
- Duration options: 1, 2, 3, or 5 days
- Hydration tracker with selectable intake levels
- Safety confirmation before starting a fast
- Full-screen active fast page (pop-out style)
- Live countdown timer based on selected duration
- Progress and water goal indicators
- Post-fast nutrition reminder message

## Tech Stack

- HTML
- CSS
- JavaScript (for countdown and dynamic values)

## Notes
